# VAC_Compiler

Proyecto de Lenguajes y Autómatas por equipo, en el que se realizó una propuesta de un lenguaje en el que cada miembro del equipo tiene que proponer 3 palabras reservadas para su verificación.

## Palabras propuestas:

### ClauVex
*   **ignore**
*   **memorize**
*   **stepto**

### ErikaHer
*   **output**
*   **stop**
*   **supersede**

### HeberPV
*   **data**
*   **enter**
* **rnumber**

### RockLee444
*   **clear**
*   **condition**
*   **inherit**
